# -
| 机床主轴 |<--| 电涡流传感器 |<--| 滤波放大器 |<--| 数据采集仪 |<--| 信号分析仪 |
|:--------:|:--:|:------------:|:--:|:----------:|:--:|:----------:|
| 被测对象 |<--| 拾振环节    |<--| 测量放大线路 |<--| 信号采集    |<--| 信号处理    |


这个系统框图的含义是：电涡流传感器将机床主轴的振动信号转换为电信号，滤波放大器对电信号进行放大和滤波，数据采集仪对电信号进行采样和转换，信号分析仪对电信号进行各种分析和处理，从而得到主轴的振动特性和状态。
